¿Como usar?

1 Selecciona la API

Busca entre todas las APIs, la que te ayude a modelar tu data rápidamente ¡El tiempo es oro!

Puedes utilizar Fetch, Axios, Request o incluso XMLHttpRequest si eres de los duros, ahora ve al paso tres.

Ya basta de procastinar, las APIs no demoran ni 1 segundo, así que manos a la obra...por cierto, este paso no deberia estar pero es para rellenar, buen provecho xD.

Preguntas frecuentes

¿APIs centralizadas?

En simple, queremos ahorrarte el tiempo de ir a buscar en múltiples fuentes la información que necesitas solo para realizar un prototipado rápido de tu aplicación web, móvil o lo que estes haciendo, por eso creemos que es necesario múltiples APIs falsas en un solo lugar a un click. Y por si te lo preguntas inspirados en JSONPlaceholder.

¿Cobrán por utilizar las APIs?

Rotundamente no, el uso de las APIs más gratis que el aire, no tienes que pagar por nada.

¿Hay limite de solicitudes?

La razón principal de las APIs falsas de esta web es que tengas data disponible rápidamente para tus proyectos, cualquier otro uso se considera inadecuado, así que limitamos las solicitudes a 10 por segundo.

¿Con que lenguajes puedo utilizarlos?

Puedes utilizar cualquier lenguage con el que puedas realizar peticiones http, eso incluye go, php, node, python, el mismo javascript desde el navegador, incluso desde tu placa de arduino o tu cafetera.

¿Cómo se genera la data falsa de las APIs?

Para generar la data, se ha utilizado como fuentes algunas librerias como faker.js, chance, y algunas apis publicas más, a estas últimas obviamente tuvimos que cambiar cierta información.

No encuentro la API que necesito

Creemos que podemos mejorar esto juntos y disponer de más APIs falsas con forma pasa el tiempo, te recomiendo apuntarte al blog, allí se publicará actualizaciones de nuevas APIs y características que se agregarán.

Las tecnologías que se utilizarón fueron Svelte para la reactividad, Bulma para estilos css y json-server para el back.
